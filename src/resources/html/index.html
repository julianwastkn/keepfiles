Early Release

What KeepFiles Does

KeepFiles is a file host and link shortener that empowers server hosts to tightly manage users, administrators, and their site in general. I built the site with the idea that this could grow into something that really has everything you need in one convenient spot. For now, I would say it's a great utility - but there's always room for improvements! Be sure to reach out about any features that you think would make this tool even cooler.

User Friendly APIs

APIs can be scary, but rest assured, they don't bite! I made them super easy to use, especially when integrating with applications like ShareX. You can find all of our documentation, including pre-made SXCU files, on our GitHub.

Open Source,
The Good & The Bad

I set out to build something useful for people that they could learn from, contribute to, and utilize in their day-to-day. This type of service is something I really desired before I knew how to program. I feel like I'm on a path to success with that original mission, and while I'm really happy about it, it does have a downside.

Because of the nature of this project, there is no financial backing. I can't afford to host the service for everyone, but you can find detailed documentation on how to get it going yourself. Check out our GitHub to learn more about the status of the project, and how you can host your own instance.
